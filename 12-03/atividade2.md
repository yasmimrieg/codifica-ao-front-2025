# Exercícios: Operadores Relacionais
## O objetivo é determinar se a expressão resulta em Verdadeiro (True) ou Falso (False).

### 1. Operador: Igual a (==)
``` 
Verifique se o resultado da operação matemática abaixo é igual ao valor comparado:

Expressão: (15 / 3) == 5

Pergunta: O resultado é Verdadeiro ou Falso?
Resposta:verdadeiro 
Explicação da resposta: é verdadeiro porque 15 dividido por 3 é 5, e == significa igual igual.
```
### 2. Operador: Diferente de (!=)
```
Analise a comparação entre as duas strings (textos) abaixo:

Expressão: "Python" != "python"

Pergunta: Considerando que a maioria das linguagens diferencia maiúsculas de minúsculas, qual o resultado?
Resposta: verdadeiro
Explicação da resposta:porque  uma comeca com letra maiuscula e outra com letra minuscula.
```
### 3. Operador: Maior que (>)
```
Imagine que um sistema de e-commerce verifica o estoque de um produto:

Cenário: estoque = 10, pedido = 12.

Expressão: pedido > estoque

Pergunta: O sistema deve bloquear a venda? (O resultado é Verdadeiro ou Falso?)
Resposta: sim, verdadeiro.
Explicação da resposta: o sistema deve bloquear a venda poris o numero de pedidos e maior do que o quanto de produto tem em estoque.
```
### 4. Operador: Menor que (<)
```
Verifique a idade mínima para uma categoria infantil de esportes:

Cenário: idade_atleta = 9.

Expressão: idade_atleta < 10

Pergunta: O atleta pode participar da categoria Sub-10?
Resposta: sim.
Explicação da resposta:porque o sub-10 e para atletas com menos de 10 anos, e como o atleta tem 9 entao ele pode.
```
### 5. Operador: Maior ou igual a (>=)
```
Um sistema de cinema concede meia-entrada para idosos (60 anos ou mais):

Cenário: cliente_idade = 60.

Expressão: cliente_idade >= 60

Pergunta: O cliente tem direito ao benefício?
Resposta: sim
Explicação da resposta:tem porque o beneficio e para pessoas com 60 anos ou mais, e o cliente tem 60 anos 
```
### 6. Operador: Menor ou igual a (<=)
```
Um sensor de temperatura dispara um alerta se o clima estiver abaixo de zero ou congelando:

Cenário: temperatura_atual = 0.

Expressão: temperatura_atual <= 0

Pergunta: O alerta será disparado?
Resposta: nao
Explicação da resposta: proque a temperatura tem que estar abaixo de 0, entao 0 ainda nao e considerado.
```
